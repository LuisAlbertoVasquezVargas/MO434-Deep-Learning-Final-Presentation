\section{Methodology}

\begin{frame}{Method Overview}
  \small
  \begin{itemize}
    \item Define the classification task and selected datasets.
    \item Train reference models under controlled experimental settings.
    \item Evaluate student or compact models using the same metrics.
    \item Compare results across datasets, architectures, and training strategies.
  \end{itemize}
\end{frame}
